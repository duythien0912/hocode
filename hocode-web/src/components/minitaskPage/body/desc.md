
Viết hàm trả về tổng của hai số.
### ví dụ
Với `param1 = 1` và `param2 = 2`, thì output sẽ là `add(param1, param2) = 3`
### Input/Output

- **[Thời gian thực thi tối đa] 3s (java)**
- **[input] integer param1**  
     *ràng buộc:*  
     `-1000 ≤ param1 ≤ 1000`  
- **[input] integer param2**  
     *ràng buộc:*  
     `-1000 ≤ param2 ≤ 1000`
- **[output] integer**  
     - Tổng của 2 input
